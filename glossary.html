Glossary

HTML- HTML (Hypertext Markup Language) is a text - based approach that describes the structure of content in an HTML file. This markup tells a web browser how the text, images and other multimedia forms can be displayed on a web page.

Headings- Element headings such as < h1 >, < h2 >, < h3 >,... Allows you to divide the document into logical sections using six levels of document headings, ranging from the largest to the smallest.

Hyperlink- When you click on it, hyperlinks (or links only) take the user to another website. Href is the most common attribute used for links, which tells the browser where the link goes.

CSS- CSS stands for Cascading Style Sheets CSS describes how to display HTML elements on screen, paper or other media

Padding- padding is the space between the contents and the border (element edge). With CSS, we can adjust this value to bring the border closer or further away from content.

Margins- margin is the element's space. The greater the margin, the greater the space between our element and its elements. We can adjust the margin to bring our HTML elements closer to each other or further away.
